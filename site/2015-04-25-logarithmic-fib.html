SICP excercises 1.19 - Logarithmic time Fibonacci number generation

2015-04-25

This exercise describes a transformation TpqT_{pq}, that, when applied to a pair (a,b)\left( a, b \right), transforms it according to a←aq+bq+apa \gets aq + bq + ap and b←bp+aqb \gets bp + aq. The transformation used to generate Fibonacci numbers, starting from the pair (0,1)\left( 0, 1 \right), can be written as a←a+ba \gets a + b and b←ab \gets a.

Note that the Fibonacci transformation is T01T_{01} (p=0p=0 and q=1q=1).

The exercise asks us to show that applying any transformation TpqT_{pq} twice (or Tpq2T^2_{pq}) to a pair (a,b)\left( a, b \right) is equivalent to applying the transformation Tp′q′T_{p'q'} for some p′p' and q′q', and to find p′p' and q′q' in terms of pp and qq.

Let’s apply the transformation once to get

$ a’ = aq + bq + ap </div><div></div> <div> b’ = bp + aq $
Now apply the transformation once more.
$ a’’ = a’q + b’q + a’p </div><div></div> <div> b’’ = b’p + a’q $
Substitute for a′a' and b′b' and rearrange
$ a’’ = ( aq + bq + ap ) q + ( bp + aq ) q + ( aq + bq + ap ) p </div><div></div> <div> = ap^2 + bq^2 + 2aq^2 + 2apq + 2bpq </div><div></div> <div> = a ( 2q^2 + p^2 + 2pq ) + b ( q^2 + 2pq ) </div><div></div> <div> = a ( q^2 + 2pq ) + a ( p^2 + q^2 ) + b ( q^2 + 2pq ) </div><div></div> <div> = a ( q^2 + 2pq ) + b ( q^2 + 2pq ) + a ( p^2 + q^2 ) $
$ b’’ = b’p + a’q </div><div></div> <div> = ( bp + aq )p + ( aq + bq + ap )q </div><div></div> <div> = bp^2 + apq + aq^2 + bq^2 + apq </div><div></div> <div> = b( p^2 + q^2 ) + a( q^2 + 2pq ) $

Now comparing the final expressions for a″a'' and b″b'' with the original transformations, we note that transforming (a,b)\left( a,b \right) by Tpq2T^2_{pq} is equivalent to the single transformation Tp′q′T_{p'q'}, where p′=p2+q2p' = p^2 + q^2 and q′=q2+2pqq' = q^2 + 2pq.
